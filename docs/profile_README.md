### b7n0de, Verified AI Work

Building the trust layer for AI work, cryptography plus verifiable AI evaluation. Human plus AI, but with proof.

- Now, **[proofbundle](https://github.com/b7n0de/proofbundle)**, a small pure Python tool that emits and verifies signed, Merkle anchored evidence bundles, and eval receipts that prove a model passed a threshold without revealing the model or the data. Selective disclosure via SD-JWT (RFC 9901) is supported on the verify side, issuance is on the roadmap.
- Focus, AI governance and evaluation, trust and verification, applied cryptography (Ed25519, RFC 6962 Merkle, SD-JWT per RFC 9901, Sigstore, SLSA, in-toto).
- Background, a mathematics and cryptography foundation and two decades of running operations heavy businesses. I turn hard technical systems into things people can check.
- Principle, a claim counts only once the code behind it actually runs.
